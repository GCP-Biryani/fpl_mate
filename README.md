# fpl_mate
`FPL Mate` - Get your team performance, mini-league performance and Watch list picks based on expected goal involvements, points per game, Form, next gameweek expected points and Infuence+Creativity+Threat rank 

# What does the script show? 
* Team details - User full name, Team name, Overall points, Overall Rank
* Team Performance
* Mini-league performance
* Watchlist (only users not in your team), based on:
    - Top-10 expected Goal involvements
    - Top-10 Points per game
    - Top-10 by Form
    - Top-10 by Next GW expected points
    - Top-10 by Influence-Creativity-Threat Index
